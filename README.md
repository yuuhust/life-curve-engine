# Life Curve Engine

**Life Curve Engine（人生曲线引擎）** 是一个面向大学生和年轻职场人的 **AI 驱动人生资源配置系统原型**。  
它不是算命产品，也不是普通的习惯打卡器、待办工具或聊天式建议器，而是尝试把每日行为输入、阶段策略与长期走势变化连接起来，形成一套 **可计算、可解释、可模拟、可建议、可验证** 的闭环系统。

---

## 项目想解决什么问题

大多数产品只能帮助用户“记录今天做了什么”，少数产品能做到“提醒你该做什么”，但很少有系统真正去解决一个更底层的问题：

> **一个人每天如何分配自己的时间、精力、健康、学习、项目与 AI 协作能力，这些选择最终会如何影响长期人生走势，以及系统是否能基于这些变化给出下一步最优动作。**

Life Curve Engine 的目标，就是把这个问题从模糊感受、经验判断和零碎记录，升级为一个结构化、可验证的产品系统。
