# Life Curve Engine

Life Curve Engine（人生曲线引擎）是一个面向大学生和年轻职场人的人生资源配置模拟器。它不是算命，也不做精确人生预测，而是把每日行为输入映射到实体状态、中间因子、曲线和解释，再形成建议、行动计划与闭环验证。

## 核心链路

```text
行为输入
→ 规则引擎
→ 实体 / 中间因子更新
→ 曲线 / 解释
→ 复盘 / 模拟
→ 建议
→ 行动计划
→ 闭环验证
→ 自然语言摘要
```

公开展示用架构图草稿见 [docs/showcase-diagrams.md](docs/showcase-diagrams.md)。

## 当前工作台主流程

1. 记录今天：填写 12 个行为因子，并选择当前阶段模式。
2. 看曲线与复盘：查看阶段优先曲线、复合曲线、周/月复盘、实体和中间因子状态。
3. 看建议与计划：根据复盘、趋势和模拟结果生成修复项、强化项和 7 天测试计划。
4. 做模拟：对睡眠、AI 协作深度、项目推进做单因子或连续 N 天模拟。
5. 看自然语言摘要：把已有结构化结果整理成复盘摘要、建议摘要和执行提醒。

首次打开且没有本地记录时，工作台会自动载入“学业优先 Demo”。也可以在“本地数据”面板手动切换到“项目优先 Demo”，用于展示不同阶段模式下的曲线、复盘、建议、模拟和摘要链路。

## 当前功能边界

- 使用本地 state 和 localStorage，不接数据库。
- 规则引擎是确定性规则，不调用真实 LLM。
- 自然语言摘要层当前是模板实现，只负责表达润色。
- AI 不参与评分、阈值判断、建议排序或计划验证。
- 阶段模式会影响复合分数权重、建议排序偏置、复盘重点和默认曲线展示。
- 暂不支持阶段自动切换。
- 暂不支持复杂多方案模拟。
- 暂不做社交、多用户、移动端完整 App。

## 主要代码结构

```text
src/app/                 Next.js 页面入口
src/components/          工作台 UI 组件
src/data/                行为因子定义
src/engine/              规则、曲线、复盘、建议、模拟、摘要等核心逻辑
src/hooks/               工作台状态编排
src/storage/             localStorage 读写与导出
src/types/               TypeScript 类型定义
docs/                    产品、架构、规则和阶段文档
```

- V2.x：增强自然语言摘要层，但继续保持只读结构化事实。
- V2.x：扩展模拟器为轻量多方案对比。
- V3：考虑接入真实 LLM，用于自然语言复盘摘要、建议润色和每日记录补全。
- V3：在不破坏核心规则链路的前提下，引入更完整的数据持久化与历史分析。
